# 今庄まち歩き PWA テスト版

GitHub Pagesの公開フォルダへ、このZIPの中身をそのままアップロードしてください。

## 主な機能
- カテゴリアイコン式TOP
- 全72スポット掲載
- 名称はExcelの文字をそのまま表示（自動ふりがななし）
- 詳しい紹介は非表示
- 検索・カテゴリ・お気に入り
- 日替わりおすすめコース
- GPS日替わりスタンプ3か所
- 観光協会QRでゴール
- PWA・オフライン対応

## テスト上の注意
- GPS・カメラはHTTPSのGitHub Pages上でテストしてください。
- GPS候補はGoogleマップURLから緯度経度を取得できた徒歩圏スポットに限定しています。
- 判定半径はテスト用に100mです。
- `goal-qr.png`を観光協会ゴール用QRとして印刷できます。
