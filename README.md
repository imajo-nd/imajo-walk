# 今庄まち歩き PWA（テスト版）

## 公開方法
このフォルダの中身をGitHub Pagesの公開リポジトリ直下へアップロードしてください。HTTPS環境で位置情報・カメラが動作します。

## 実装済み
- Excelの72スポットを一覧表示
- 詳しい紹介は表示しない
- 検索・カテゴリ絞り込み
- 日付を種にした日替わり3スポット
- GPS半径70m以内でスタンプ取得
- 3スポット達成後、観光協会ゴール案内
- 観光協会QR読取でコンプリート
- PWAインストール・オフラインキャッシュ

## 重要：GPS対象
Excel内のGoogleマップURLに緯度経度が含まれているスポットのうち、今庄宿徒歩圏だけを日替わり候補にしています。現在は10件前後です。全スポットを候補にするには `data/spots.json` の各スポットへ lat / lng を設定し、stampEligible を true にしてください。

## ゴール用QR
QRコードの内容は `IMAJO-GOAL-2026-V1` です。同梱の `goal-qr.png` を観光協会に掲示してください。

## テスト時の注意
ブラウザの位置情報許可が必要です。iPhoneではSafari、AndroidではChromeでの確認を推奨します。QR読取はブラウザ対応状況により、カメラまたは画像選択を利用します。
